# 📚 BookWarm - Backend

This is the **backend service** for **BookWarm**, an online book café application that allows users to browse, order books, and enjoy a café-like experience virtually.  
Built with **Spring Boot**, **MySQL**, and **JWT-based authentication**, the backend is designed to be **scalable, secure, and production-ready**.

---

## 🚀 Features
- **User Authentication & Authorization**  
  - Secure login and signup using **JWT** and **Spring Security**.
- **CRUD Operations**  
  - Manage books, orders, and user profiles.
- **Role-Based Access Control (RBAC)**  
  - Separate access for users and admins.
- **Optimized APIs**  
  - Clean architecture and efficient queries for faster response times.

---

## 🛠 Tech Stack
- **Java 17**
- **Spring Boot**
- **Spring Security with JWT**
- **MySQL**
- **Hibernate/JPA**
